### VFD - OFF-Line(Standby) USV

- Voltage Frequency Dependant
- nicht für server
- wenn schalter im schaltbild vorhanden, ist eine offline usv

### VI Line interactive

- für kleinere server
- minimale umschaltzeit

### VFI On-Line USV

- keine umschaltzeit
- für größere Server
- ist immer an
